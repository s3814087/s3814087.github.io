WHY IT?

Im interested in IT because i want to learn how to read and write source code so I can understand why software behaves in a particular way and also so I can build applications myself.

This interest started when i got a job at a fintech company, where I still currently work as a product specialist. When there is an issue with the software that we cannot explain,we will get a developer to grab logs and then check the source code to determine why something is behaving in a particular way. This is particularly common for issues that we cannot replicate ourselves.

Watching the developers solve these complex issues over time has given me an appreciation for the skills required to do this and it's now a skill I want to learn myself so I can eventually become a developer and triage these issues.

WHY RMIT?

I currently work full time and previously had tried studying part time at another uni in the past, but the physical commute to that uni after work and late night commute home after lectures and tutorials was quite exhausting. This uni also did little to help students learn in my opinion.

After this I started looking for online courses so I could do the lectures and tutorials on my own time at home and eliminate the burden of commuting at night. RMIT was the only uni in australia offering an online IT degree so i signed up for one subject to try it out.

I quickly found that RMIT was very focused on making learning as easy and convenient as possible for students and the semester schedule would allow me to study a full time workload at a part time rate with the 4 semester system, so i enrolled in the degree and haven't looked back.
